 TROUBLESHOOTING:


Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "No module named 'video_processor'"
Solution: Make sure you run from frontend/ folder

Problem: "API key not found"
Solution: Create frontend/.env with your API keys

Problem: "Port 5000 already in use"
Solution: Edit app.py line 167, change port to 5001

Problem: "Database error"
Solution: Delete frontend/sop_generator.db and restart
